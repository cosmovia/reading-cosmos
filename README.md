# 🌌 Reading Cosmos Pro MAX

> Build your own reading universe.

Reading Cosmos Pro MAX 是一个 AI 驱动的阅读管理系统。

它诞生于一个非常简单的问题：

> 当阅读发生在手机、电子阅读器、纸质书、图书馆等不同渠道时，我们如何能够轻松记录所有阅读，并真正把知识沉淀下来，而不是停留在「读过」？

Reading Cosmos 希望成为这个答案。

它不仅帮助你管理读过的书，更希望帮助你构建属于自己的知识宇宙，让每一本书、每一个观点、每一次思考彼此连接，并最终形成持续成长的认知网络。

# 📖 Reading Cosmos 阅读方法

Reading Cosmos 并不追求记录得越多越好，而是希望用更轻量、更适合不同类型书籍的方式帮助阅读真正发生。

因此，根据不同类型的书籍，采用不同的阅读记录方式。

---

## 🛠 工具类书籍

采用 **第一性原理式阅读法（First Principles Reading）**。

围绕三个核心问题展开：

- **What Problem（解决什么）**
  > 这本书试图解决什么问题？

- **With What（用什么工具）**
  > 作者提出了哪些方法、模型或工具？

- **Why This Tool（为什么有效）**
  > 为什么这个方法能够起作用？它背后的第一性原理是什么？

相比记录大量内容，更关注理解底层逻辑。

---

## 📚 小说 / 人文 / 非虚构作品

采用 **R-A-T 内化笔记法**。

### R · Resonate（共鸣）

记录最触动自己的：

- 人物
- 情节
- 概念

回答：

> 谁触动了我？

---

### A · Abstract（提炼）

提炼：

- 三个最重要的思想
- 值得长期记住的金句

回答：

> 它告诉了我什么普世真相？

---

### T · Transform（转化）

记录：

- 阅读后的启发
- 可以实践的行动

回答：

> 阅读之后，我会变成一个怎样不同的自己？
